# Ball Analysis

A professional basketball analytics dashboard and player CRM. Tracks every in-game statistic in real time — deflections, miles run, quarter-by-quarter performance, shot charts, hockey assists, screen assists, loose balls, box outs — and pairs it with a CRM for workouts, meal plans, rest tracking, and physical measurements.

## Features

- **Live tracking** — court visualization with real-time player/ball positions and a quick action recorder for advanced stats
- **Games** — box scores, shot charts, play-by-play, and team efficiency summaries
- **Players & CRM** — season averages, workout plans, meal plans, recovery tracking, and physical measurements (stride length, wingspan, standing reach, vertical leap)
- Responsive layout for desktop and mobile

## Tech Stack

- [Next.js](https://nextjs.org) with Turbopack
- [Tailwind CSS](https://tailwindcss.com)

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) — it redirects to the dashboard.
